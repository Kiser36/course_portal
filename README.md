 INSTRUCTIONS
 . Database Configuration
Open phpMyAdmin (http://localhost/phpmyadmin).
Create a new database named ise_cms.
Click the Import tab
Select the ise_cms.sql file (included in this repository/folder).
Click Go to generate the tables and sample data.

3. File Deployment
Download the repository/ZIP file.
Place the project folder (named courseportal) into your server's root directory:
For XAMPP: C:/xampp/htdocs/courseportal
Open db.php (or your connection file) and ensure the credentials match your local setup:
Host: localhost
User: root
Password: "" (empty)

4. Launching the System
Open your web browser and enter the following URL:

http://localhost/courseportal/index.php
